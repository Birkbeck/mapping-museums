Folder content

* basic_stats: This folder contains simple statistics for all museums (allmus_) and
just for museums open in 2017 (mus2017_). These stats focuses on single variables.

* geodemo_analysis: This is the geodemographic analysis that includes urban/rural,
deprivation, and output area classification.

* maps: Cartography

* museum_visits: museum size estimation

* openclose: Openings/closures data and analysis

* size_govern_subj: This folder contains simple statistics for all museums (allmus_) and just for museums open in 2017 (mus2017_). These stats combine two variables.
